# NYC Rent Stabilization Lookup

Tenant-facing web app that answers two questions:

1. **Is my building rent-stabilized?** (BBL lookup against the NYCDB
   `rentstab_v2` snapshot.)
2. **Given my lease history, am I being overcharged?** (RGB Apartment
   Order walk-forward; 6-year HSTPA statute window.)

Plus a one-click draft of [DHCR Form RA-89](https://hcr.ny.gov/form-ra-89)
generated by Claude.

See [Design.md](./Design.md) for the full architecture and phase plan.

## Quickstart

```bash
npm install
cp .env.example .env.local           # add ANTHROPIC_API_KEY for the complaint draft
# place data/seed/rentstab_v2.csv   (see data/README.md for the source URL)
npm run seed                         # builds data/app.db (~50K+ buildings)
npm run dev                          # http://localhost:3000
```

## Environment

- `ANTHROPIC_API_KEY` — required for `/api/complaint`. The lookup and
  estimate endpoints work without it.
- `APP_DB_PATH` — optional override; defaults to `./data/app.db`.
- `GEOSEARCH_BASE_URL` — optional; defaults to NYC Planning Labs public Pelias.

## Deploying to Vercel

`better-sqlite3` is a native module, so it has to be configured as an
external server package. That's already set in [next.config.ts](./next.config.ts).
Two deploy concerns:

1. **The seed.** `data/app.db` is gitignored. For Vercel, either:
   - commit the built DB (simplest, but the file may be large), or
   - check in the source CSV under `data/seed/` and let `npm run seed`
     run during the build (`vercel.json` is configured to do this).
2. **Function timeout.** The `/api/complaint` route streams from
   Anthropic, which can take longer than the default 10s Hobby cap.
   The route exports `maxDuration = 60` so it gets the full 60s.

Set `ANTHROPIC_API_KEY` in the Vercel project's environment variables.

## Data sources

| Source | Used for | Where it lives |
|---|---|---|
| NYCDB `rentstab_v2` | Stabilized unit counts + DHCR list flag, by BBL | local SQLite (seed) |
| RGB Apartment Orders #1–#57 | Annual % increases | local SQLite (seed) |
| NYC Planning Labs GeoSearch | Address → BBL + lat/lng | runtime, no key |
| Anthropic API (Claude Sonnet 4.6) | Complaint draft | runtime, requires key |

## Disclaimer

Not legal advice. Estimates only. The tool does not model MCI/IAI
adjustments, vacancy allowances, or fraud claims. Always verify with
DHCR and consider speaking with a tenant attorney before filing.
